# DEFINITIVE LOGO COLLECTION v2
## Pastel Retro Terminal Design System

All logos use the same template structure as RegexGPT:
- 60-character width double-line border
- Block letters for product name
- Connected small boxes for accent/subtitle
- Consistent padding and spacing

export function ProductLogo({ product, className = '' }) {
  const config = LOGOS[product];
  if (!config) return null;
  
  return (
    <pre 
      className={`text-xs leading-tight ${className}`}
      style={{ color: config.color }}
    >
      {config.art}
    </pre>
  );
}

# IMPORTANT NOTES

1. **Escape backslashes** in JavaScript: The ship wheel uses `\\` for backslashes

2. **Font**: Requires monospace font with box-drawing support (JetBrains Mono recommended)

3. **Background**: Designed for dark background `#1a1a2e`

4. **Line height**: Use `leading-tight` or `line-height: 1.2`

5. **File encoding**: Must be UTF-8
